# run localy without docker . 

use the following command to run the project localy without docker:
1. git clone https://github.com/KevynKrancen/HomeTask.git
2. cd HomeTask
3. npm install
4. change the local configuration in the server.js "const PORT = 3000" to the port you desire ( eg : const PORT = 3001);
5. npm start
6. upload files and enjoy.
